Simulation Options

Normally, the simulation will generate a new random path every n seconds. Turn this off to have it stop.


By default, no minimum path distance is set. You can change that here. Longer paths take longer to find.

Since the tiles are picked at random, the simulation will eventually give up and produce a shorter path, if it is taking too long.
