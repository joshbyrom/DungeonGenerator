Simulation Options

Normally, the simulation will generate a new dungeon every n seconds. Turn this off to have it stop.


By default, the number of rooms in the dungeon are small. You can change that here. More rooms means more processing time.

Since the room locations are picked at random, the simulation will eventually give up and produce a smaller dungeon, if it is unable to find spaces to place all the rooms.
